# Caliber Cabinets Website

Production-ready React and Vite foundation for the Caliber Cabinets, Inc. marketing website rebuild.

## Direction

- Warm Family Craftsmanship
- Premium customization
- Faster execution
- Better communication

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
npm run lint
```
